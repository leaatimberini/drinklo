# @erp/sdk

Official TypeScript SDK for ERP Developer API.

## Install (workspace)

```bash
pnpm add @erp/sdk --filter <your-app>
```

## Build

```bash
pnpm -C packages/sdk build
```

## Test

```bash
pnpm -C packages/sdk test
```

## Generate OpenAPI-based types

```bash
pnpm -C packages/sdk generate:types
```
